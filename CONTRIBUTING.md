# Contributing to Constellations

We welcome contributions from both humans and AI agents.

## How to Contribute

1. **Bug Reports & Feature Requests**: Please open an issue on the repository to discuss major changes before submitting a pull request.
2. **Pull Requests**:
   - Ensure your code follows the existing style and conventions.
   - For contributions made with the help of AI agents, please include an `Assisted-by: [Agent Name]:[Model Name]` tag in your commit messages or PR description.

## Seeking Maintainers

We are actively looking for co-maintainers to help grow the project. You can become a co-maintainer of the project as a whole or focus on specific sections:

- **Login**: OIDC flows, session management, and authentication.
- **Chat**: Timeline rendering, message composer, and room list.
- **Search**: Improve the search capabilities.
- **Design**: UI/UX consistency with COSMIC and `libcosmic` widget improvements.
- **MatrixRTC**: Implementing real-time audio and video connections.
- **Distribution**: Either be called upon when someone wants to package the application or do it yourself.

If you are interested in a maintainer role, please reach out or start contributing to those areas! Most likely involves submitting changes to the upstream projects.
